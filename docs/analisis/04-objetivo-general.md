# 04 — Objetivo General

## Objetivo General
Desarrollar un sistema de información web responsivo que permita a los estudiantes
de la Corporación Universitaria Remington localizar aulas, consultar horarios académicos
y calcular rutas de navegación dentro del campus, integrando un mapa interactivo,
un asistente virtual y un módulo de notificaciones, con el fin de mejorar la experiencia
estudiantil y reducir los tiempos de desplazamiento interno para usuarios nuevos.

## Alineación con el Problema
| Problema identificado | Solución que aporta el sistema |
|---|---|
| Estudiantes no encuentran sus aulas | Mapa interactivo + búsqueda por código/docente |
| Falta de integración horario-ubicación | Módulo académico vinculado al mapa |
| Sin notificaciones ante cambios de aula | Sistema de notificaciones internas |
| Navegación ineficiente por el campus | Cálculo de rutas con tiempo y distancia estimados |
| Sin registro de consultas previas | Historial de búsquedas por estudiante |
