# 03 — Alcance del Sistema

## Dentro del Alcance

### Funcionalidades incluidas
- Autenticación de usuarios (Estudiante, Docente y Administrador)
- Visualización de mapa interactivo del campus (torres, pisos, aulas)
- Búsqueda de aulas por código, docente o materia
- Cálculo y visualización de rutas hacia aulas
- Consulta de horario académico personalizado
- Registro de historial de consultas
- Sistema de notificaciones internas
- Asistente virtual para consultas en lenguaje natural
- Reporte de fallos técnicos y seguimiento de estado
- Panel de administración para gestión de horarios, docentes y aulas

### Usuarios cubiertos
- Estudiante (actor principal)
- Administrador del sistema
- Equipo de desarrollo (soporte técnico)

### Tecnología cubierta
- Aplicación web responsiva (PC, tablet, móvil)
- Base de datos relacional SQL
- Comunicación segura mediante HTTPS

## Fuera del Alcance

- Aplicación móvil nativa (iOS / Android)
- Integración en tiempo real con sistemas externos de la universidad
- Nave
